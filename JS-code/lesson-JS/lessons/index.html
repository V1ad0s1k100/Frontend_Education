Заметки по JS

  • Циклы

    Для перебора свойств объекта используется цикл for-in. Он проходит по всем перечислимым свойствам объекта.

    Для перебора итерируемых объектов, таких как массивы, строки, Map, Set и т.д. используется for-of

    Цикл for используется для итерации с известным количеством повторений

    Цикл while выполняется до тех пор, пока условие истинно

    Цикл do-while выполняется хотя бы один раз, а затем повторяется, пока условие истинно
